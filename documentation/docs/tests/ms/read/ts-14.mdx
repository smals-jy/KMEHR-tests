## Description

See if software in test can handle all the authorized [CD-TEMPORALITY](https://www.ehealth.fgov.be/standards/kmehr/en/tables/temporality) by [MS Cookbook](https://www.ehealth.fgov.be/standards/kmehr/file/cc73d96153bbd5448a56f19d925d05b1379c7f21/057e64396c1ea312a0b01607b0f94e4adcf456e8/20210331-safe_cookbook_medicatieschema_v5.8_en.pdf) :

## Test cases

In other words, only a subset of [CD-TEMPORALITY](https://www.ehealth.fgov.be/standards/kmehr/en/tables/temporality) is authorized by [MS Cookbook](https://www.ehealth.fgov.be/standards/kmehr/file/cc73d96153bbd5448a56f19d925d05b1379c7f21/057e64396c1ea312a0b01607b0f94e4adcf456e8/20210331-safe_cookbook_medicatieschema_v5.8_en.pdf) & my experiences :
    - "acute"
    - "chronic"
    - "oneshot"

Anything else (e.g. "ifnecessary") is rejected by vaults

## Test cases

In other words, all values of [CD-ADMINISTRATIONUNIT](https://www.ehealth.fgov.be/standards/kmehr/en/tables/drug-administration-unit), including these ones that were prohibited by [MS Cookbook](https://www.ehealth.fgov.be/standards/kmehr/file/cc73d96153bbd5448a56f19d925d05b1379c7f21/057e64396c1ea312a0b01607b0f94e4adcf456e8/20210331-safe_cookbook_medicatieschema_v5.8_en.pdf) but now authorized in KMEHR 1.40.2 :
    - ampoule
    - bandage
    - bottle
    - box
    - effervescent-tablet
    - iu
    - liter
    - meq
    - micrograms
    - miu
    - mmol
    - piece
    - syringe
    - syringe-ampoule

## Expected results 

When looking to the patient medication scheme, for each value authorized :
    - The medication must be present (for easiness, they all have daily periodicity)
    - The administration unit translation to humans should respect [CD-ADMINISTRATIONUNIT](https://www.ehealth.fgov.be/standards/kmehr/en/tables/drug-administration-unit) table

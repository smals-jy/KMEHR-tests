## Description

See if software in test can handle all the authorized [CD-PERIODICITY](https://www.ehealth.fgov.be/standards/kmehr/en/tables/periodicity) by [MS Cookbook](https://www.ehealth.fgov.be/standards/kmehr/file/cc73d96153bbd5448a56f19d925d05b1379c7f21/057e64396c1ea312a0b01607b0f94e4adcf456e8/20210331-safe_cookbook_medicatieschema_v5.8_en.pdf) :

```xml title="Example"
...
<transaction>
    ...
    <item>
        <id S="ID-KMEHR" SV="1.0">2</id>
        <cd S="CD-ITEM" SV="1.11">medication</cd>
        ...
        <frequency>
            <periodicity>
                <!-- The focus on the test -->
                <cd S="CD-PERIODICITY" SV="1.0">D</cd>
            </periodicity>
        </frequency>
    </item>
    ...
</transaction>
...
```

## Test cases

In other words, all values of [CD-PERIODICITY](https://www.ehealth.fgov.be/standards/kmehr/en/tables/periodicity) except these ones prohibited by [MS Cookbook](https://www.ehealth.fgov.be/standards/kmehr/file/cc73d96153bbd5448a56f19d925d05b1379c7f21/057e64396c1ea312a0b01607b0f94e4adcf456e8/20210331-safe_cookbook_medicatieschema_v5.8_en.pdf) :
    - ondemand
    - UE
    - UN
    - UQ
    - US
    - UX
